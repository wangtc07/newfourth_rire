京都府出身
　
　
　
弓木奈於（ゆみきなお）です。( ¨̮ ) ( ¨̮ ) ( ¨̮ )
　
　
　
　
　
　
　
　
ブログのコメントでピオーネ（ぶどう）をおすすめしてくださったので、早速食べました✨
　
　
　
ほっぺたが落ちるとはまさにこの事！と思ったほど美味しかったです☺️
　
　
　
おすすめしてくださってありがとうございます(*´˘`*)♡
　
　
　
　
　
　
　
　
　
　
すごくすごく嬉しい事
一番最初に書いてもいいですか˙˚⸜(* ॑ ॑* )⸝˚˙
　
　
　
　
　
8月9日（日）18:00～18:30
文化放送さんで放送される
　
　
　
乃木坂46の「の」
　
　
に出演させていただきます！！！✨
　
　
　
　
　
乃木坂46の「の」に出演させていただけることを知った時、本当に本当に嬉しくて
岩本蓮加さんとしっかりお話しさせていただけるのも初めてだったので、
　
　
「あと『5日』っ☺︎」
　
「あと『3日』っ☺︎」
　
　
とワクワクしながら毎日を過ごしていました(∩ˊᵕˋ∩)･*
　
　
　
　
　
　
　
　
　
　
　
　
　
　
　
ピクニックか！！
　
　
　
　
　
　
　
　
　
　
　
　
\(ᯅ̈ )/  ﾁｶﾞｳｰ
　
　
　
　
　
　
　
　
　
　
　
　
どうしよう、いっぱいお話ししたい...！
でも放送はまだなので、言えない...！
　
　
　
岩本蓮加さん、綺麗で本当に優しくて
お話しさせていただいている時もずっと幸せの中でした✨
私自身、私も知らなかった新たな一面なので是非聴いてくださったら嬉しいです( ˃ ˂ )✨
　
　
　
本当に楽しくて幸せな1日でした☺︎✨
　
　
　
　
　
　
　
　　　　　---------- ｷﾘｶｴ ---------
　
　
　
　
　
　
　
❁少しお返事させていただきます❁
　
　
　
⚫︎ 今までコスプレはやった事ありますか？
　
　
コスプレと言えるのかはわからないのですが、小学生の頃ハロウィンパーティーの仮装で、魔女やウサギや女性ヒーローキャラクターの格好をした事があります！
　
　
　
⚫︎ 最近よく話す4期生は？
　
　
清宮レイちゃんはこの間4.5時間？ぐらいずーっとお話ししていました☺︎
レイちゃんの包容力？温かさに溢れていて一緒にいる時すごく落ち着きました☺︎
レイちゃんお誕生日おめでとう✨
可愛くて、優しくて、いつも一生懸命なレイちゃんが大好きです♡
素敵な一年になりますように...✨
　
　
　
　
⚫︎今好きなお笑い芸人さんは誰ですか？？
　
　
いっぱいいっぱいいらっしゃって...
　
バナナマンさんはずっと大好きで、さらば青春の光さんも姉と盛り上がるほど大好きで、東京03さん、オードリーさん、ハライチさん、土田晃之さん、渡辺直美さん、サンドウィッチマンさん、くりぃむしちゅーさん、千鳥さん、ティモンディさん、ジャルジャルさん、ネプチューンさん、四千頭身さん、NON STYLEさん、ミキさん...書ききれぬほど本当にいっぱいいらっしゃいます...！
　
　
　
　
⚫︎ いま欲しいものは～？
　
　
電子レンジとお洋服、欲を言えば魔法の力(•'-'•)
　
　
　
　
⚫︎寝る前に必ずする事とかある？？
　
　
まつ毛美容液を塗って、鏡に布をかけます！
　
　
　
　
⚫︎ 僕は怒ってしまうと何も言わない派なのですが
ずっと怒りっぱなしで1日が終わります......
何が切り替え方法とか対処法みたいなのありますか？
　
　
実は私あまり怒らなくて...怒ったとしても数分で忘れちゃうんです...
あ！姉はお家で「ほんまなんなんや〜〜〜♪」って大きな声で歌いながらステップ踏んで移動していました！
　
　
対処方法を調べてみたら、
・体を動かす　・好きな音楽を聴く　・好きなアロマをたく　・瞑想をする　・お風呂に入る
などがありました...！
　
ぴったりな方法探索のお手伝いが出来ていたら有り難いです(＞＜)
周りの事を考えて自分の中で整理しようとなさっているお姿を聞いて、とても優しいお方で素敵ですっと思いました！
毎日大変な中、本当にお疲れ様です✨
　
　
　
　
　
　
皆さんテスト、お仕事...
毎日本当にお疲れ様です( ˃ ˂ )✨
　
　
　
そして、『ノギザカスキッツ』の感想もコメントしてくださってありがとうございます( ᐪᐤᐪ )
　
　
昨日の放送は、
「ホワイトエンジェルの災難」
　
4人とも可愛いくて面白くてテレビの前から離れられませんでした...✨
　
「やんちゃなやんちゃん(2)」
　
軽快なリズムは頭から離れなくて、「やんちゃ〜♪」と何度も頭で流れています（笑）
紗耶ちゃんも、賀喜ちゃんのツッコミも、ツッコム時の表情も面白くて大好きです✨
　
　
　
勉強！！
　
　
　
　
　　　　　---------- ｷﾘｶｴ ---------

でしたっ！笑
　
　
　
可愛いか微妙なぬいぐるみって逆に気になりますね☺︎笑

可愛くて綺麗で、そして瑠奈ちゃんらしい配信でずっと心が跳ねていました☺︎
　
「のぎおび選手権」お話ししながらの46秒は難しそうと見ていて思いました( ᐪᐤᐪ )！
　
　
　
今日は黒見ちゃんの配信ですね✨
心の底っから楽しみです(◦ˉ ˘ ˉ◦)♡
　
　
　
　
　
　
　
むーにー！
　
　
　
　
最後まで読んでくださりありがとうございました( ˃ ˂ )

本日もありがとうございました☺️
