2020/0915Tue新4期生リレーわたし。佐藤璃果







こんにちは~




乃木坂46 （新）4期生  岩手県出身




佐藤璃果です。




太陽が顔半分にあたって

光と影に別れているのが好きなお写真です。









"本日は｢EX大衆10月号｣さんの発売日"








有難いことにマネージャーさんが

沢山撮ってくださったので

沢山オフショットをお送りします~






雑誌で初めて前髪を分けました。





新鮮ですよね

私も新鮮です。





メイクさんやスタイリストさんが

好きだと言ってくださった歯磨き。




今までに無い新しい私なので、

みなさんがどう受け取ってくださるか

ドキドキですが、楽しみです( *´﹀`* )




今回の撮影で前髪なしの自分に対しての

拒否反応が前よりもなくなって、感謝です！


皆さんが暖かくてお優しくて、


楽しかったです。






夏らしい事をするのが難しい今年の夏に

花火をするという素敵な思い出をくださって

ありがとうございました。






目をつぶっちゃってるけど

花火を皆さんにお届けしたいので(^^)






本誌の方では赤いワンピースで

目がちゃんと開いている写真があるので

そちらを見てください笑













ちらり。|˙꒳​˙）




オフショット好評だったらまた載せようかな︎︎︎︎︎☺︎






~~~~~~~~~~~~~~~~~~~



ファンレター読ませて頂きました！



沢山愛がぎゅっと詰まっていて

本当に本当に嬉しかったです。


何故か、みなさん、｢重くてすみません｣

と書いていたんですが、


それぐらい全然重くないですよ！！

愛の深さだなと重い嬉しくなります。




私もっと重い手紙送ってしまいますもん。

 



研修生事務局宛のものもあり、

その時の私たちにも、ちゃんと

応援して下さっていた方がいたんだなと

心がほっとしました。




"心が通じる"素敵な時間でした。




ありがとうございます。






~~~~~~~~~~~~~~~~~~~


~~~質問~~~




○ 【質問】というか気になったんだけど、

「新内さん足綺麗てん」の"てん"って、「です」か何かの誤字なのかそれともそういう表現があるの？



すみません！！ものすごく誤字です。

もう1回言いますね、


新内さん足綺麗...


が正しいです。


申し訳ないです。











○ ミスドのね、おいもシリーズすっごく美味しそうで。りかちゃんは食べた〜？




食べれていません（ ; ; ）

私も、ものすごく気になっていました。



もし食べていたら、

どれがオススメなのか情報待ってます♡










○ 乃木坂駅に坂道研修生のポスターがあって調べたらブログが出てきたので読ませていただきました！可愛いですね！応援してます






なんて素敵な。

ポスターを作成して下さった方、

見つけて下さった方も

ありがとうございます！！



ポスターを作って下さったんですか...。

感謝です。本当に心の底から。

乃木坂駅のポスター憧れでした。

15人で憧れを実現出来たかと思うと


感謝です。本当に心の底から。


新たな出会い、嬉しいです。

よろしくお願いします。









○ りかちゃんって何か癖ってありますか？

僕はペットボトルやコップを持つとき、小指が立ってしまうんですヨ(笑)



待ってください、、私もです、、、（ ; ; ）

一緒だぁ~











○ 璃果ちゃんは、しんどいなって思う時どういう曲を聴きますか？



本当にしんどい時は曲も聞けなくなって

しまうかもしれないんですが、


最近は

笑顔   /  いきものががり さん


裸の心  / あいみょん さん


JAM / BiSH さん


リズム / BiSH さん


stereo future / BiSH さん



聞いてみてください( *´﹀`* )



どうか自分を大事になさってくださいね！

一緒に乗り越えていきましょう









○人間関係が上手くいかない時璃果ちゃんはどうしてますか？



私もずっと苦手なので

私にとって永遠の人生のテーマです。（笑）


どういった場合なのか詳しい事は

分からないのでなんとも言えないけれど、


今の時代色んな所に

コミュニティーがあるので、


そこの人間関係が上手くいかなかったら、

別のコミュニティーで元気を出すとか、？


例えば私のブログに沢山コメントするとか


できるだけ返します！



人間関係上手くいかない時のきっかけって

人それぞれだから難しいですね、、


力になれず申し訳ない、、


抱え込まないで下さいね（ ; ; ）


いつか大丈夫になる日が来ますきっと。




❁




前回のコメント欄に沢山おすすめの

お洋服教えてくださって

ありがとうございます！！









~~~~~~~~~~~~~~~~~~~





そして「ノギザカスキッツ」のコント

かつ家で、念願の瑠璃ペアで

幼なじみかつをさせて頂きました。



林瑠奈と幼なじみで制服着れるなんて。。。




電話で練習しあったり相談したのも

嬉しかったです、



明日はそんな林瑠奈のブログです


らじらー！サンデー

お疲れ様〜- ̗̀ ♡  ̖́-







~~~~~~~~~~~~~~~~~~~





それでは

皆さんにとって

素敵な１日になりますように。







撮影の時にして下さった

素敵なマニキュア♡






またね。






#29 りか
