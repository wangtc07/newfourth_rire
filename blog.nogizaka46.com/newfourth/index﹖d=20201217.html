乃木坂46 4期生

京都府出身

弓木奈於（ゆみきなお）です☺︎



昨日は、レコメン！さんに出演させていただきました( *ˊᵕˋ ) ⁾⁾

のりさん、真佑ちゃん、やんちゃん、とたくさんお話しさせていただけて

本当に嬉しかったです✨

クイズが出来たり、

私の拙い言葉を笑いに変えてくださって、優しい言葉で包んでくださって本当にありがたい気持ちでいっぱいです( ˃ ˂ )✨

笑顔をいっぱいくださった

素敵な3時間を、ありがとうございました✨

　

弓木あるある。

知らぬ間に手にインク付きがち( ˙ᒡ̱˙ ) 

ボールペンでも、なんでも

メモした後は必ずと言っていいほど手にインクがついています

なんで〜

記念に2枚☺︎


　　　　　---------- ｷﾘｶｴ ---------

❁少しお返事のコーナー❁

⚫︎ なおちゃんは遊園地で絶叫系乗れますか！！！

はい！！

( ˙-˙ )←こんな顔で乗れます

⚫︎ 朝起きるのが辛すぎて、寝坊しそうになるんだよね〜

何かいい方法ありますか？

床暖房をつけたり...朝美味しい朝ごはんを用意したら起きられそうではなかろうか...？(⊃'-'∩)ｼｭｯ

⚫︎ ユミッキーは小さい頃サンタさんからプレゼント貰いましたかー？

はい！

実は今もいらっしゃいます（小声）

お勉強、心から応援しています✨

⚫︎クリスマスプレゼントは何が欲しいですか？

オーブンレンジが欲しいです⸜(* ॑ ॑*)⸝ﾖｸﾊﾞﾘ

⚫︎ この前の舞台『ナナマルサンバツ』のYouTube配信ではなおちゃんクイズは最下位だったけど、お稽古では答えられるようになってきた？

それがですね...

徐々に、徐々に答える事が出来るようになってきています( ｰ̀֊ｰ́ )✧

クイズの楽しさを知って、勉強の楽しさも感じているので、楽しみにしていてください☺︎

《自分でハードルあげてるのか...？》

⚫︎ 奈於ちゃんは雪だるまを作ったことはありますか？

はい、あります！

冬になって雪が降るとすぐに、兄弟や両親と一緒に作っていました(∩ˊᵕˋ∩)･*

家の中に持って入ったら怒られて、泣く泣くベランダに飾ったりしていました☺︎笑

もうすぐクリスマスなんですね！

早い( ⊙⊙)!!

コメント、すごくすごく嬉しいです✨

いつもありがとうございます✨

掛橋沙耶香ちゃんと

「ノギザカスキッツ ACT2」

についてお話しさせていただきました✨

まるで会話が聞こえてくるかのようです！

是非よろしくお願いします( *ˊᵕˋ ) ⁾⁾

堀未央奈さんの大切な曲を皆さんとご一緒させていただける事感謝申し上げます✨

が日本武道館にて開催なさいます✨
