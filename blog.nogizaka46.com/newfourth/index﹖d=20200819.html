2020/0819Wed新4期生リレー改めて☺︎弓木奈於

こんにちは！
　
　
　
ページをクリックしてくださりありがとうございます☺️
　
　
　
　
乃木坂46 新4期生
　
　
　
京都府出身
　
　
　
　
　
弓木奈於（ゆみきなお）です\( ¨̮ )/
　
　
　
　
　
　
　
　
　
　
　
最近家具の位置に迷っています(•'-'•)
　
　
　
　
〝窓の近くにベッドを置くのはどうなんだろう...〟
　
　
　
〝ここに机置いたら角気になるな...〟
　
　
　
〝くぼみ気になるな...〟
　
　
　
などなど
　
　
　
　
出来るだけ、素敵な空気を取り入れられるようなお部屋にしたいな〜って思っているので
　
　
まだまだ試行錯誤は続きそうです¯\( ˘-˘ )/¯
　
　
　
　
　
　
　
　　　　　---------- ｷﾘｶｴ ---------
　
　
　
　
　
　
　
　
　
　
　
　
　
　
これはなんぞや...？
　
　
　
　
　
　
　
これは、ブランケットです！
　
　
　
　
　
　
頭の部分に手を入れると手も温まって、触り心地がふわふわで気持ち良いです(∩ˊᵕˋ∩)･*
　
　
　
　
　
　
　
ジャケットを着ている友達や家族のフードの下に手を入れて温まったりしたことありませんか...？
　
　
　
　
　
　
　
　
(｡･-･｡) ｱﾚ?
　
　
　
　
　
　
　
私は、仲良しの友達や家族のフードを見つけると「あ✨」ってなります☺︎
　
バレないように、そーろと入れようとするのですが...
　
　
　
すぐバレちゃってました( ˙༥˙ )笑
　
　
　
　
　
　
　
　
　　　　　---------- ｷﾘｶｴ ---------
　
　
　
❁質問のお返事コーナー❁
　
　
　
　
　
⚫︎ゆみっきー怖いの得意なのかなぁ？
　
　
怖いの多分大丈夫だと思います！
怖い話の話し方で
「そ〜れ〜は〜......あなただーーー！（大声）」
って急に大声を出す手法を初めて受けた時はびっくりしましたが
「そ〜れ〜は〜...」って前触れがある事に気がついたので、多分大丈夫だと思います！
　
　
　
　
⚫︎ 北野日奈子さんの写真集とか持ってる？
　
　
はい！もちろん持っています☺︎
先輩方の写真集は見えるところに大切に置いています✨
　
　
　
　
⚫︎ 弓木ちゃんは最近ハマってる食べ物とかありませんか〜？
夏バテ気味で食欲がなくなってきてて何を食べれば良いのかわかりません( ˘~˘ )
　
　
フルーツトマトや、梨、すだちにハマっています！
夏バテ本当にお辛いのにいつもコメントありがとうございます( ›_‹ )
夏バテには、豚肉や梅干しやスポーツドリンクなどが良いそうです...！
学校や課題にいつも頑張っていらっしゃって、私にも温かい言葉をかけてくださって本当に素敵で温かい方ですと思っています✨
早く元気になりますようにと心から祈っております。どうかお大事になさってください( ›_‹ )
　
　
　
　
⚫︎奈於ちゃんはかき氷で好きな味とかありますか？
　
　
ブルーハワイが好きです！
かき氷屋さんに行って、ふわふわのいちご練乳や、フルーツや白玉がのっているかき氷など、食べてみたいかき氷がいっぱいあります˙˚⸜(* ॑ ॑* )⸝˚˙ 
　
　
　
　
⚫︎ 辛い料理は食べれますか？
　
　
辛みに弱い方だと思います...！
辛い調味料をかける事がかっこいい！と思って
私も我が物顔でかけてしまった時、非常に後悔しました。笑
　
　
　
　
⚫︎梅干しはなんで50秒レンジでチンするんですか！？
　
　
梅干しを加熱するとクエン酸と糖分が結合して、血流をよくして代謝を活発にする物質が出来るとテレビで見て
やってみるとすごく好きな味だったので、大好きになりました☺︎
　
　
　
　
⚫︎ なおちゃんは一人カラオケとか一人旅行等の一人◯◯は何処まで可能ですか？
　
　
経験した事があるのは、
1人歩き、1人過ごし、1人カフェ、1人乗車
とかですかね...
経験は少ないですが、どこまでも大丈夫な気がします☺️
　
あ、でも1人海水浴とかはちょっと寂しいですかね( ˃ ˂ )
1人森探索とか、1人無人島も厳しいですかね...
そう考えたら意外とありますね(´._.｀)
　
今経験したいと思っているのは、1人焼肉、1人出前です！
　
　
　
　
⚫︎ 京都のお土産で、千枚漬けを頂いた事があるのですが、弓木ちゃんは漬け物好きですか？
　
　
はい！
祖母が千枚漬けや大根の甘酢漬けやぬか漬けを作ってくれているので、食卓に必ずあります☺︎
大好きです✨
　
　
　
　
⚫︎ 弓木ちゃんは何色のメイクをすることが多いですか？
　
　
ベージュのような茶色が多いです！
たまに薄いピンクをベースにピンクがかったブラウンのグラデーションに挑戦したりします|•'-'•)و✧
　
　
　
　
⚫︎ 載せれるのであれば弓木さんの昔の写真とか見たいです
　
　
いっぱい写真を撮ってくれる両親だったので、家にアルバムが大量にあります（笑）
　
　
　
母によると、幼稚園の時のお餅つき大会の時みたいです！口がえらい事になってますね（笑）
　
　
　
　
⚫︎バーベキューは出来た？
　
　
バーベキューは出来てないです( ˃ ˂ )
最近誰かから「お家でバーベキューした！」って聞いたような...妹ですかね？（たぶんそう笑）
　
　
　
　
⚫︎唐突な漢字クイズ！野菜の名前です。
1.西瓜　2.木耳　3.甘藍
　
　
1.すいか　2.きくらげ　3.かんろ
　
私にはわかります。3は確実に間違っていることを...
　
　
調べてきます( '-' )ง✧笑
　
　
　
　
⚫︎ 一番好きな野菜はなんですか？（ちなみに私はじゃがいもです）
　
　
一番か...
レタスとおなす！（2つ...笑）
じゃがいも美味しいですよね(*ˊᵕˋ*)♪
　
　
　
　
　
　
　
　
　
暑い中、課題やお仕事や学校や習い事など皆さん本当にお疲れ様です( ›_‹ )✨
　
　
起きた事とかお話ししてくださってとても嬉しいです☺️
　
　
　
　　　　　---------- ｷﾘｶｴ ---------
　
　
　
明日は黒見ちゃんです！
　
　
　
　
昨日の美佑ちゃんのブログに元気をもらいました☺︎✨
　
　
　
　
あ、昨日林瑠奈ちゃんに
　
〝るなち〟
　
と連絡すると
　
　
　
〝どうしたーーーーーーーー！！！（大声）〟
　
　
ってボイスメッセージが来て
笑いが止まりませんでした☺︎笑
　
　
予想の遥か上を行く林瑠奈です。
　
　
一気に元気になりました☺︎
　
　
　
　
　
　
　
　
　
　
暑い日が続いていますが、お身体に気をつけて素敵な一週間をお過ごしください( ›_‹ )✨
　
　
　
　
　
皆さんからいただいたコメントを読んですごく心の底から嬉しかったんです！（ずっと毎日です！）
　
　
皆さんにも嬉しい事が沢山起きる素敵な一日になりますようにと心から願っております✨
　
　
　
　
　
今日ブログを書かせていただいたり、
こうして皆さんとコメントを通してお話し出来る事が本当に幸せで、有難いことだといつも感じています。
　
　
　
　
改めて
　
皆さんに、楽しんでいる姿や挑戦している姿や成長していく姿など沢山の姿をお見せ出来るように
少しでも「嬉しいな☺︎」って思っていただけるように
「ありがとう」をお伝え出来るように
しっかり自分を掴めるように
一生懸命頑張りたいです。
　
　
　
　
　
これからも、
よろしくお願いします☺︎✨
　
　
　
　
本日もありがとうございました☺️
