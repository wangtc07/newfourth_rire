2020/0908Tue新4期生リレーよくやってしまうピース☺︎弓木奈於

こんにちは！
　
　
ページをクリックしてくださり、ありがとうございます☺️
　
　
　
乃木坂46 新4期生
　
　
　
京都府出身
　
　
　
弓木奈於（ゆみきなお）です( ˙༥˙ )
　
　
　
　
　
　
　
　
　
今まで史上、最大の斜めですね（笑）
　
　
　
　
　
　
　
　
最近、『進撃の巨人』を見ました( ¨̮ )
　
　
　
　
『進撃の巨人』をご紹介なさっている番組を見て、
　
「そんな展開なのか！？」
「えー！？」
　
とまさかまさかの展開だったので
気になって、少しずつですが見ています。。
　
　
　
　
いっぱい頭を使ったり、記憶力も大切で、
　
「もしも私がその世界の住人だったら、今と同じ考えなのかな...」
　
と考えたりもします。
　
　
　
　
　
でも何事にも考え始めたら、えらい深くまで考えてしまうのが私の注意点なので...笑
　
本人（主人公）よりも考えすぎないように注意します。笑
　
　
　
　
　
　
　
　　　　　---------- ｷﾘｶｴ ---------
　
　
　
　
　
『ノギザカスキッツ』
昨日放送でしたね✨
　
　
　
　
「麗しのレイランド様」
　
　
レイランド様と森田さん、レイランド様と東ブクロさんの掛け合い大好きで、ずっと笑っていました☺︎
　
レイランド様を取り合う2人も可愛すぎました♡
　
店員さん役の瑠奈ちゃんのコールも、コールをすると瑠奈ちゃんに聞いた時からずっと楽しみで
初めて見た時はすごく嬉しかったです！！
「グイグイ、グイグイ〜」が頭から離れません（笑）
　
本当に面白すぎました。大好きです。
　
　
　
　
　
「妄想少女・かっきー」
　
　
個人的に「おもしれー女」という台詞が出てきて嬉しかったです！笑
賀喜ちゃんの一人二役、本当に面白くて可愛くて
キュンキュンしました(∩ˊᵕˋ∩)･*
　
セーラー服の黒見ちゃんとあやめちゃんの会話も可愛いかったです\(ᯅ̈ )/♡
　
　
　
　
　
　
　
　　　　　---------- ｷﾘｶｴ ---------
　
　
　
　
❁お返事のコーナー❁
　
　
　
　
⚫︎なおちゃんは今年夏らしいことしましたか？？？
　
　
お家でかき氷を食べました！( '-' )ง✧
　
　
　
　
⚫︎何の秋が好き〜？？
　
　
私も食の秋が好きですԅ(¯﹃¯ԅ)
今はかぼちゃの煮物が食べたいです...！
洋服の秋も好きです！
友達と行った紅葉のライトアップも綺麗でした☺︎
　
　
　
　
⚫︎ コーヒーのブラックは飲めますか？
　
　
飲めないです（小声）
これから飲めます（小声）
　
　
　
　
⚫︎秋に食べたいものは？
　
　
焼き魚！
塩焼き！
　
ԅ(¯﹃¯ԅ)
　
　
　
　
⚫︎ 弓木ちゃんは秋の味覚といったら何ですか？？
　
　
梨、ぶどう、かぼちゃ、さつまいも！
さんま！！
　
ԅ(¯﹃¯ԅ)
　
　
　
　
⚫︎ サインの由来とか解説があったら興味ある！
　
　
由来！！
最初は漢字がいいな〜と思っていたのですが、漢字は繋げるのがなかなか難しくて...
ローマ字だとnaoで短くて...
ひらがなで「ゆみきなお」だと簡潔で、伸ばせるところを伸ばして繋げたらサインが出来上がりました！笑
　
　
　
　
⚫︎ カレーには福神漬けですか？らっきょうですか？
それ以外ですか？
　
　
福神漬けをのせていただきます(∩ˊᵕˋ∩)･*
　
　
　
　
⚫︎ 欅坂46さん守屋麗奈ちゃんとの思い出やエピソードがあれば教えてほしいです。
　
　
今でも連絡取ります〜☺︎✨
お見せできる写真がなくて申し訳ないです( ›_‹ )
優しくて可愛くて、とても話しやすい存在です✨
坂道研修生ツアーの時に麗奈ちゃんが持っていた芋けんぴが、私がすごく行ってみたいなと思っていたお芋屋さんだったので盛り上がりました。笑
地方に住んでいた事もあってお出かけは出来なかったのですが、いつかお出かけしたいです☺︎
　
　
　
　
⚫︎小学生の時、クラスで何係りだったか覚えてる？
　
　
何係りってありましたかね...？（記憶力）
音楽委員で、トランペットやアコーディオンやティンパニや木琴を弾いたり
放送委員で音楽をかけたり
図書委員をしていた気が...します...（記憶力）
　
　
　
　
⚫︎ 浴衣姿で出かけるならどこへ行きたいですか？
　
　
京都では、お着物や浴衣を着ていらっしゃる方がたくさん歩いていらっしゃって
そのお姿を見るたびに「素敵だな〜✨」と思っていました！
私も浴衣を着てお出掛け、お散歩、してみたいです☺︎
来年絶対しよう！そして皆さんにお見せしたいなって思います( ¨̮ )♡
　
　
　
　
　
　
　
免許の結果や、日常のお話、お仕事の嬉しい報告などお話ししてくださって本当にありがとうございます！
そして心から、おめでとうございます✨✨
いつも優しい言葉をありがとうございます...( ᐪᐤᐪ )
　
　
　
　
私も皆さんと一緒に前向きに頑張りたいです！✨
　
　
　
　
　
　　　　　---------- ｷﾘｶｴ ---------
　
　
　
　
　
　
　
9月6日は吉田綾乃クリスティーさんのお誕生日でした✨
　
　
　
いつもご挨拶をすると、ふわふわな優しい笑顔でお返事してくださって
「お優しい...有り難い...( ᐪᐤᐪ )」
といつも思っています。
お洋服もオシャレで、いつも目で追ってしまいます...✨
いつかお話ししたいな...って思います( ›_‹ )♡
素敵な一年になりますようにと心から願っております✨
改めて、お誕生日おめでとうございます！
　
　
　
　
　
　
　
　
　
　
明日は黒見ちゃんです！
　
セーラー服姿可愛すぎました(੭*ˊᵕˋ)੭♡
　
　
　
　
　
美佑ちゃん、人狼しようね！！！
めっちゃ久しぶりだよね！！！
　
　
　
そして、あなたが呼んでくれている「なお〜」で皆さんも呼んでくださる事があるんだけど
なんだか幸せな気持ちになるよ。ありがとう。笑
　
　
　
　
　
　
　
最後まで読んでくださり、ありがとうございました✨
　
　
　
　
　
　
　
　
久しぶりの三つ編みでしたっ☺︎

本日もありがとうございました☺️
