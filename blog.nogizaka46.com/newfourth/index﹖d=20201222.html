乃木坂46 4期生

京都府出身

弓木奈於（ゆみきなお）です( ¨̮ ) ( ¨̮ ) ( ¨̮ )

アンダーライブ2020、3日間本当にお疲れ様でした✨

私は、配信LIVEで拝見しました✨

ずっとずっと楽しみにしていた日で、

正座待機とはまさにこの事という状態でした。

ノンストップで次々と変わってゆく世界の中で

一曲一曲繊細に違う表現に何度も鳥肌が立ちました。

あまりに美しくて、輝いていらっしゃって、私の表情筋は緩むばかりでした。

　

アンコールの前にお客様の映像も拝見したのですが背中からでもファンの皆さんの熱が伝わってきました。

先輩方皆さんがライブに込めていらっしゃる熱い気持ちや、お互いを支え合っていらっしゃって、温かい想い合いに溢れていらっしゃる姿

会場の一体感に

思わず涙してしまいました。

　

改めて、自分自身ももっともっと頑張りたいと思いました。

様々な色のPowerをお届けしてくださって、ありがとうございました✨

本当に、本当に

素敵なライブを、ありがとうございました。✨
