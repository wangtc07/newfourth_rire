前回のブログでは、
粕汁とクッキーの作り方を発表させていただきました☺︎


「大丈夫かな...？」

とドキドキだったのですが、

「作ってみたい！」と言ってくださったり

「作ったよ！」と報告してくださったり
とってもとても嬉しかったです！( ･ ･̥ )✨




いつもありがとうございます☺️



　　　　　---------- ｷﾘｶｴ ---------



今日は、坂道グループ合同研修生ツアーのお話（前編）をしたいと思います❁



初めてのブログのコメントから、坂道グループ合同研修生ツアーのお話をたくさん書いてくださり、本当にありがとうございます(﹡ˆᴗˆ﹡)




昨年の秋頃
東京、大阪、愛知、３つの都道府県で、Zeppさんのステージに立たせていただきました。




初めて、坂道グループ合同研修生ツアーを開催していただけるというお話を聞いた時


言葉にできないほどの感謝の気持ちと同時に
不安な気持ちがありました。

そして何よりも
先輩方やたくさんの皆様が大切にしていらっしゃった曲を踊らせていただく事が大変恐縮で申し訳なく思いました。




だがしかし！



研修生として活動させていただく時に
「自分から変わらないといけない。」
と心に誓ったのです。



だから私がやるべきことは不安になることじゃなくて、一生懸命練習すること
だと思いました。



「申し訳ない...」
という考えを

「〇〇って思っていただけるように頑張ろう！」

に変えました。




ダンスの先生は、
ダンスが未経験の私にダンスの覚え方からステップまで丁寧に教えてくださったり

先輩方のパフォーマンスを何度も何度も見たり、
歌詞について考えたり
15曲、1曲1曲に込められた想いがあって、身体のどんな小さな一部の角度がほんの少し違うだけで受ける印象が違ったり、
『表す』ことの難しさを知りました。


初めての事ばかりで、右も左も分からぬ私達に一から丁寧に教えてくださいました。


いつも支えてくださっているたくさんの方と

15人全員で、
皆様にお会いできる事を本当に楽しみに
泣いたり笑ったり、かけがえのない夏を過ごさせていただきました。



いよいよ本番
舞台袖で手を握りしめ合っていた私達の耳に
見に来てくださった皆様のお声が届いた時は
緊張と不安とワクワクと感謝と、様々な気持ちが織り混ざって
心が熱くなりました。



ステージ上では、


曲によってサイリウムの色を変えてくださったり

まさか、私たちの名前を書いてくださっている布類や紙類などを作ってきてくださるなんて
想像もしていませんでした。



キラキラ輝く素敵な景色や、温かいお言葉や優しい笑顔をたくさん
本当にありがとうございました( *ˊᵕˋ ) ⁾⁾ ❁





私の名前もしっかり見えていました☺️✨

バスケットボールのマークをつけてくださったり、黒字で名前を書いてくださったり、
名前を呼んでくださったり
とってもとっても嬉しかったです(*ˊᵕˋ*)♡
本当にありがとうございました☺️




あの日見た景色や、かけてくださったお言葉は今でも心に大切に保管しています✨
（ちゃんとロックもしてあるんですゾウ）




反省ばかりですが、
研修生として活動させていただけて
ほんの少しですが1年半前と考え方がまったく違います。
15人で、マネージャーさんやたくさんの皆様に教えていただき、支えていただきながら過ごした時間は本当にかけがえのない大切な時間です。


先輩方や、今まで坂道グループを築いていらっしゃったたくさんの皆様のお力で
経験させていただける事に感謝の気持ちを忘れずに


教えていただいた事や学んだことをちゃんと底に持って、これからたくさんのことを吸収できるように頑張ります！




( '-' )ง✧

先日、
『乃木坂46時間TV アベマ独占放送「はなれてたって、ぼくらはいっしょ！」』
の開催が発表になりましたね(﹡ˆᴗˆ﹡)

乃木坂46時間TVに参加させていただける事が信じられないほど光栄で有り難く、とても嬉しいです( ›_‹ )✨



少しでも、皆さんが笑顔になっていただけるように精一杯頑張ります！


皆さんと楽しい時間が過ごせたらいいな( *ˊᵕˋ ) ⁾⁾

〝規則正しい生活を送ってそう〟

皆さんにとって素敵な一週間になりますように...( ⁎ᵕᴗᵕ⁎ )
