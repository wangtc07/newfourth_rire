2020/0612Fri新4期生リレー思い出冷凍保存。佐藤璃果






こんにち~☺︎



今日も暑いですね~

6月12日華の金曜日！



乃木坂46 新4期生 岩手県出身


佐藤璃果です。



この4コマ、どこからスタートでどこが

ゴールとかさっぱりで、悩みました(´._.`)




そうしたら

数学の象限の話を思い出したので、

第1象限から第4象限の順でお願いします。

(いつの話＆なつかしい)



①パンを食べて、②何撮ってるの、

③まぁ、食べるけど、④おいしいよっ！


というストーリーです。



この日は人生初！のクリスマスパーティー

というものを開いた日です。(もう夏くるよ)




パーティーを開いたのも初！あれ、

2人だったらパーティーって言わないのかな




友達を家に呼ぶのも数年ぶりで

緊張していました、、




私は彼女が来るのを家で、待っていました。




キッシュを焼いて、冷蔵庫でケーキを冷やしておいたり、サラダの上の卵はハートや星の型抜きをしちゃったりして、、、



ちなみに！

キッシュ作りはちゃんと1回練習しました~





彼女は可愛いので、私も写真をパシャパシャ撮りました♡眼が幸せです♡

可愛いを50回ぐらいは一日で言っていそう。




可愛い子好きはるなぴと同じですね( ˙ᵕ˙ )




るなぴごめんよ、

いつも可愛いって言ってくれるのに、

｢もうまたそうやって~！｣とか言ってしまい、、


可愛い！と言われると

気になってしまうのです、、


好き！と言われると、

気になってしまうのです、、



ただ照れてるだけです、嬉しいです！



ちょっと特別に見えてきて

他の子にもデレデレしてるのを見るとちょっと悲しくなるといいとますか


難しい！


でも、みんな可愛いから当たり前なんです！

全員本当に可愛くて、可愛すぎて、

目が合わせられません。

みんなかわいい。

皆さんの可愛さに追いつけるぐらいに

可愛くなりたいです。




るなぴもかわいいです。



照れて写真に逃げる

最近佐藤璃果って打とおとすると

佐藤璃かじっている。

となってしまいます。

まさにこの写真でしょうか(´._.`)




今日はわたあめのような

ふわふわとした気持ちで書いてしまった


美味しいですか？

そんな私。me.

影がすごい。




*****************************************



乃木坂46時間TV、

4期生の期別企画に参加させていただきます。


なんと企画が、

"みんなでお菓子の家を作ろう"


とても素敵です、ありがとうございます。


4期生、新4期生の"みんなで"

そして、見てくださってるかた"みんなで"


夢みたいな、心の中で描いていた

絵本の1ページを作れる事、幸せです。


これを機に、もっと4期生の皆さんと沢山お話できるといいなと思います。

できるかな？

いつお話しようか何のお話をしようか

タイミングを伺う私が目に浮かびますね。


楽しんで頂けるように、私も楽しんで、

素敵な夢の世界を作れたらいいなと思います。


見てくださったら嬉しいです。



*****************************************




そしてそして、

｢ノギザカスキッツ｣に新4期生も

今後、参加させていただきます❁︎



楽しみですね。



コントは全くしたことが無いので

どきどきです。

沢山学び、沢山皆さんに楽しんで

頂けるように努めます( ੭ ˙˙ )੭



よろしくお願いします。



6月16日（火）から

毎週火曜午前1：29～1：59

日本テレビさんにて放送です。



*****************************************



6月8日、筒井あやめさん、

お誕生日おめでとうございます( ˶˙ᵕ˙˶ )


素敵な一年になりますように！❁




*****************************************




~~~質問~~~


〇好きな絵文字や顔文字を教えてください！！


最近ハマっている絵文字

になってしまいますが、

( ˊ꒳ˋ ) ᐝ (｡-_-｡) (  ˊ࿁ˋ ) (*¨*)♡


辺りです。





〇毎日コメントしてるんだけどブログ更新の次の日とかのコメとかも読んでくれてますか？



毎日読んでいますよ。

いつも本当にありがとうございます。







〇髪染めてみたいって思いますか？（もしくは今まで髪染めたことありますか？)

あとはヘアケアでダメージケアやヘアオイル等何かしてたりしますか？



染めても大丈夫な所だったのですが、

なんやかんや1度も染めていません。


インナーカラー憧れていました。


無難に茶髪もいいですし、ピンクブラウン系とか、アッシュグレーにインナーカラーピンクとか紫とか可愛いなぁと思います。



多分、しばらく染めません、



ヘアオイル、毎朝毎晩塗っていますよ！





〇秋田県に来た事ありますか？



もちろんあります！

お隣さんですので。






〇最近大学始まってまた忙しくなりそうでりかちゃんに会えなくて寂しいから応援してほしいな笑



これから皆さん忙しくなるのですね。。

お疲れ様です！


毎日、何か一つでも、

楽しいことや嬉しいことがありますように。






〇璃果ちゃんは占いとかは興味がありますか？

 


大好きです。

毎日のようにチェックしてしまいます。

毎週月曜日は、しいたけ占いさんを絶対見てしまいます。


いつか占い屋さんに行きたいのですが、

行けていません。




〇身長測りたいって言ってたけど

何かしらの形で測れた??



メジャーを足で踏んで自分で測ってみようとしたんですが、上手く行きません(; ;)




*****************************************



明日はるなぴです。

私も林瑠奈ちゃんをお好きな方と

お話してみたいですよ(ᯅ̈ )


#林瑠奈ちゃん推しとお話したい


使ってみたかったんです。


*****************************************


皆さんにとって今日が

素敵な一日になりますように。






巻き巻きにやり(¯ｖ¯)ﾆﾔ


またね。
